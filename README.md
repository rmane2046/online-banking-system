Project Title: Online Banking System (Java)

Description:
This project is a simple console-based Online Banking System developed using Core Java.
It allows users to create bank accounts, log in, view balance, transfer funds, and view
transaction history. The project focuses on backend logic and object-oriented programming concepts.

Features:
- Create new bank account
- User login authentication
- View account balance
- Fund transfer between accounts
- View transaction history
- Console-based menu system

Technologies Used:
- Java (Core Java)
- OOP concepts
- Collections Framework

How to Run:
1. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)
2. Compile all Java files
3. Run the Main.java file
4. Follow the menu options in the console

Note to keep in mind:
This project is developed as part of a Java Internship task and focuses mainly on backend
functionality.
